Lista de Verificación para la Configuración de Litd

- [ ] Cuando se compila litd desde la fuente, necesitaremos tanto GoLang como Nodejs
- [ ] Instalar una versión reciente de GoLang
  -- $ wget https://golang.org/dl/go1.21.0.linux-amd64.tar.gz
  -- $ sudo tar -xvf go1.21.0.linux-amd64.tar.gz
  -- $ sudo mv go /usr/local && rm go1.21.0.linux-amd64.tar.gz
  -- $ mkdir ~/go
- [ ] Agregar la ruta de Go
  -- $ nano ~/.profile
-- Agregar:
  -- GOPATH=$HOME/go
  -- PATH="$HOME/bin:$GOPATH/bin:$HOME/.local/bin:/usr/local/go/bin:$PATH"
- [ ] Recargar el perfil
  -- $ . ~/.profile
- [ ] Comprobar que GoLang fue instalado
  -- $ go version
- [ ] Instalar Nodejs
  -- $ sudo apt install -y curl
  -- $ curl -fsSL https://deb.nodesource.com/setup_22.x -o nodesource_setup.sh
  -- $ sudo -E bash nodesource_setup.sh
  -- $ sudo apt install -y nodejs
- [ ] Instalar Yarn
  -- $ sudo npm install -g yarn
- [ ] Comprobar que npm, yarn y NodeJS fueron instalados y es una versión estable
  -- $ node -v
  -- $ npm -v
  -- $ yarn --version
- [ ] Clonar el repositorio lightning-terminal
  -- $ git clone https://github.com/lightninglabs/lightning-terminal.git
  -- $ cd lightning-terminal/
  -- $ git checkout v0.14.0-alpha
  -- $ make install
  -- $ make go-install-cli
- [ ] Comprobar que litd fue instalado
  -- $ which litd
- [ ] Crear un archivo lit.conf
  -- $ mkdir ~/.lit
  -- $ nano ~/.lit/lit.conf
- [ ] Ver el archivo de ejemplo lit.conf en /example-files/lit.conf
- [ ] Crear una billetera LND, una contraseña y un archivo wallet_password
  -- $ mkdir ~/.lnd
  -- $ touch ~/.lnd/wallet_password
  -- $ openssl rand -hex 21 > ~/.lnd/wallet_password
- [ ] Ejecutar litd
  -- $ litd
- [ ] A través de un proceso separado -otra terminal-, crear una billetera LND
  -- $ lncli create
- [ ] Seguir las indicaciones y usar la contraseña generada como la contraseña de la billetera
- [ ] Generar una nueva semilla
- [ ] Omitir el cifrado de la frase semilla (¡O cifrarla si eso tiene sentido para su caso de uso!)
- [ ] ¡Respaldar apropiadamente su frase semilla! ¡¡¡MUY IMPORTANTE!!!!
- [ ] Descomentar las siguientes dos líneas en el archivo lit.conf
  -- lnd.wallet-unlock-password-file=/home/ubuntu/.lnd/wallet_password
  -- lnd.wallet-unlock-allow-create=true
- [ ] Ver el archivo de ejemplo litd.service en /example-files/litd.service
- [ ] Configurar litd para que se inicie al reiniciar a través de systemd
- [ ] Crear un archivo en /etc/systemd/system/
  -- $ sudo touch /etc/systemd/system/litd.service
[ ] Habilitar y ejecutar el servicio
  -- $ sudo systemctl daemon-reload
  -- $ sudo systemctl start litd
[ ] Comprobar que litd se está ejecutando como se espera
  -- $ sudo systemctl status litd
  -- $ sudo lsof -n -P -i | grep LISTEN
